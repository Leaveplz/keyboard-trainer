# Тренажер клавиатуры МШП

Статический тренажер для 2 класса.

## Режимы

- **Буквы**: русские строчные и заглавные буквы, включая `ё`.
- **Знаки**: пунктуация, математические знаки, `@`, `<`, `>`.
- **Числа**: только цифры.
- **Фразы**: по одному предложению на выбранную тему.

## Настройки

- время раунда;
- скорость появления;
- один шарик или несколько шариков;
- движение шариков снизу вверх или неподвижные шарики;
- только строчные буквы или строчные вместе с заглавными;
- штраф за неправильную клавишу;
- звук.

В режиме фраз есть кнопка подсказки `?`, которая показывает нужную клавишу или сочетание клавиш для следующего символа.
