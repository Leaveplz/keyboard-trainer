# Тренажер клавиатуры

Статический тренажер для младших школьников: русские буквы с чередованием регистра, знаки, числа, символы английской раскладки и отдельный режим предложений для тренировки пробелов и пунктуации.

## Запуск

Откройте `index.html` в браузере. Для GitHub Pages достаточно загрузить в репозиторий файлы:

- `index.html`
- `styles.css`
- `app.js`

## Режимы

- **Буквы**: строчные и заглавные русские буквы, включая `ё`.
- **Знаки**: пунктуация и математические знаки.
- **Числа**: цифры, а также `@`, `<`, `>`.
- **Фразы**: предложения с запятыми, тире, кавычками и правильными пробелами.
