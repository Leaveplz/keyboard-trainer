# Тренажер клавиатуры МШП

Статический тренажер для младших школьников. Его можно разместить на GitHub Pages без сборки и серверной части.

## Файлы для публикации

- `index.html`
- `styles.css`
- `app.js`
- `assets/logo.png`

## Режимы

- **Буквы**: русские строчные и заглавные буквы, включая `ё`.
- **Знаки**: пунктуация и математические знаки, которые можно набрать с клавиатуры.
- **Числа**: цифры, а также `@`, `<`, `>`.
- **Фразы**: предложения для тренировки пробелов и знаков.

## Настройки

- время раунда;
- скорость появления;
- один шарик или несколько шариков;
- движение шариков снизу вверх или неподвижные шарики;
- штраф за неправильную клавишу;
- звук.

В режиме фраз есть кнопка подсказки `?`, которая показывает нужную клавишу или сочетание клавиш для следующего символа.
